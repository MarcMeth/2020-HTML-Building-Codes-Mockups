National Building Code of Canada 2020, Printing #3

Part 1 Compliance

Section 1.1. General

1.1.1 Application of this Code

1.1.1.1. Application of this Code

  1. This Code applies to the design, construction and occupancy of all new buildings, and the alteration, reconstruction, demolition, removal, relocation and occupancy of all existing buildings. (See Note A-1.1.1.1.(1).)
  2. This Code applies to both site-built and factory-constructed buildings. (See Note A-1.1.1.1.(2).)
  3. Farm buildings shall conform to the requirements in the National Farm Building Code of Canada 1995.
